# BMI-Calculator

The program shall prompt the user to enter weight in pounds and height in inches. An individualâ€™s BMI shall be calculated inside method getBMI() and method getStatus() returns the appropriate results i.e: Underweight, Normal e.t.c. 

**Formula**

BMI Formula = weight * KG_PER_POUND / (height * METERS_PER_INCH *height * METERS_PER_INCH)

KG_PER_POUND=0.45359237
METERS_PER_INCH=0.0254

**BMI Status**

Less than 18.5 => Underweight

Between 18.5 to 24.9 => Normal

Between 25 to 29.9 => Overweight

30 or more than 30 => Obese
